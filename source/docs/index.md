# Привет! 👋

Добро пожаловать на мой сайт-портфолио, где я публикую все свои лабораторные работы.

## Что здесь есть?

Этот сайт создан с помощью **MkDocs** и темы **Material**, поэтому он обладает крутыми возможностями:

| Фича | Описание |
|------|----------|
| 📚 **Навигация по вкладкам** | Все разделы (`Главная`, `Об авторе`, `Семестры`) вынесены в верхнее меню |
| 📂 **Секции** | Лабы удобно сгруппированы по семестрам |
| 📑 **Оглавление справа** | У каждой страницы есть содержание, которое помогает быстро найти нужный пункт |
| 🔍 **Поиск** | Можно искать по всему сайту — работает быстро |
| 🖍️ **Подсветка кода** | Код в отчётах подсвечивается, его удобно читать |
| 📋 **Копирование кода** | У каждого блока с кодом есть кнопка `копировать` |
| 🌙 **Светлая/тёмная тема** | Переключай в меню (кнопка в виде солнца/луны) |
| 🔗 **Всплывающие ссылки** | Удобные подсказки при наведении |

## Как пользоваться?

1. В меню выбери **Лабораторные работы**
2. Пользуйся ссылками из таблицы или из бокового меню
3. Читай цель, задание, код и выводы
