## Ссылка на GitHub: [Репозиторий с кодом](https://github.com/jUmb0-juiceee/Python_ITMO_moded/tree/main/sem2/LR_2)

## Описание задачи

NumPy — это основная библиотека для численных вычислений в Python. В этой работе нужно было освоить базовые операции с массивами: создание, изменение формы, транспонирование, векторные и матричные операции, статистический анализ и визуализацию. В качестве данных использовался файл students_scores.csv с оценками студентов по математике, физике и информатике. Задача состояла в том, чтобы реализовать все функции в main.py так, чтобы они проходили тесты из test.py. Код должен был соответствовать PEP-8, содержать аннотации типов и докстринги.

## Ход работы

**1. Создание структуры проекта.** Создал структуру проекта: папки data/ и plots/, файлы main.py и test.py.

**2. Загрузка данных.** В файл data/students_scores.csv добавил таблицу с оценками десяти студентов по трём предметам.

**3. Создание массивов.** Реализовал базовые функции для создания массивов: create_vector() через np.arange(10) и create_matrix() через np.random.rand(5,5).

**4. Преобразование массивов.** Написал функции для изменения формы массива (reshape_vector) и транспонирования матрицы (transpose_matrix).

**5. Векторные операции.** Реализовал векторные операции без циклов: сложение векторов, умножение на скаляр, поэлементное умножение и скалярное произведение через np.dot.

**6. Матричные операции.** Сделал матричные операции: умножение матриц через оператор @, определитель через np.linalg.det, обратную матрицу через np.linalg.inv и решение системы линейных уравнений через np.linalg.solve.

**7. Загрузка датасета.** Для работы с данными реализовал load_dataset — загрузку CSV через pandas с преобразованием в NumPy массив.

**8. Статистический анализ.** Написал statistical_analysis, который считает среднее, медиану, стандартное отклонение, минимум, максимум и перцентили. Использовал np.mean, np.median, np.std, np.percentile.

**9. Нормализация данных.** Реализовал normalize_data — Min-Max нормализацию по формуле (x - min) / (max - min).

**10. Визуализация.** Для визуализации сделал три функции: plot_histogram — гистограмма оценок по математике; plot_heatmap — тепловая карта корреляции предметов через seaborn; plot_line — линейный график зависимости оценки от номера студента. Все графики сохранялись в папку plots.

**11. Тестирование.** Запустил тесты через pytest. Все тесты прошли успешно.

## Выводы

NumPy позволяет работать с массивами без явных циклов, что ускоряет вычисления и упрощает код. Векторизация — главный принцип работы с NumPy, она заменяет циклы на операции над целыми массивами. Функции для статистики и линейной алгебры уже реализованы в библиотеке, их не нужно писать с нуля. Аннотации типов и докстринги помогают понять код и упрощают отладку. Правильная структура проекта с отдельными папками для данных и графиков делает работу организованной. Pytest — удобный инструмент для автоматической проверки кода, он сразу показывает, какая функция работает неправильно. В целом, NumPy — это база для машинного обучения и анализа данных, и без неё ничего не работает.
